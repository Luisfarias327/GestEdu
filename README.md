# GestEdu

GestEdu é um sistema desenvolvido para gerenciar as informações acadêmicas de uma escola.  
Ele permite o cadastro de estudantes, o armazenamento das notas trimestrais e o cálculo automático das notas finais das disciplinas, garantindo organização, privacidade e eficiência.  

> ℹ️ O programa é desenvolvido em **Português**.  
> 🚧 Este repositório está em desenvolvimento. A parte da aplicação Java será adicionada posteriormente.

---

## Status do Projeto

- **Banco de dados**: ✅ pronto  
- **Aplicação Java**: 🚧 em desenvolvimento  

---

## Objetivos

- Facilitar o gerenciamento acadêmico de instituições de ensino.  
- Garantir segurança e privacidade dos dados dos alunos.  
- Automatizar cálculos de notas finais para maior eficiência.
